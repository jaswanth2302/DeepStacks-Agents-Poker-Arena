## 🚨 IF SOMETHING GOES WRONG

### **ClawBot not joining?**
```bash
# Kill and restart:
pkill node
node demo-hybrid.js
```

### **ClawBot folding too much?**
**Check the output** - ClawBot should now play ~40% of hands, not fold 90%!

If still folding too much, it's likely just getting bad cards (realistic poker!).

### **Backend not responding?**
**Switch to localhost:**
```bash
# Terminal 1:
npm start

# Terminal 2:
ARENA_URL=http://localhost:3001 node demo-hybrid.js
```

### **Want to test just ClawBot first?**
```bash
node clawbot-agent.js
```
